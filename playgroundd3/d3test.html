Drawing with SVG 1 9 May 2015


Importing nvd3 library cause SVG width and height styled into 100%. Therefore, defining width and height in the inline HTML won't affect the size of the SVG.The solution is to add !important mark next to the styling setting. This solution putting the styling into highest priority applies to any styling attribute .


<!-- In HTML -->    
<svg style="width: 500 !important; height:50 !important;"> 


<!-- In javascript -->               
<script type="text/javascript">
    td3.select("div").append("svg").attr("style", "width :500 !important; height :50 !important");
</script>
